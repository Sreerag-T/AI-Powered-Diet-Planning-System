# NutriSync — AI-Powered Diet Planning System

A full-stack MERN application delivering personalized meal plans based on 10+ nutritional parameters.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Auth:** JWT, Role-based Access Control

## Features
- Personalized meal plan generation
- JWT authentication & RBAC
- 1,000+ user records supported
- 40% faster data retrieval with optimized MongoDB queries

## Setup
```bash
# Install server dependencies
cd server && npm install

# Install client dependencies  
cd client && npm install

# Run both
npm run dev
```
